## kongyo-ts-react-doctor

One command scans your codebase for security, performance, correctness, and architecture issues, then outputs a 0–100 score with actionable diagnostics.あくまでも参考程度の推奨事項に過ぎない点に留意し、修正等を実行する場合は良く吟味して必要な場合のみ行うこと。

### 実行

```bash
npx -y react-doctor@latest . --verbose
```
